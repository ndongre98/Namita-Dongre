moola
Python, Flask, MongoDB, IBM Watson, CSS

Objective:

Moola is a stock sentiment analysis tool. For each company that is queried, moola scrapes relevant articles from financial experts and then uses IBM Watson's NLP API to perform a sentiment reading on these articles.

Motivation:

moola was a hackathon project that I did with some of my closest friends in 2017. We built it with the goal of quantifying the sentiment associated with a company in order to provide people one more data point when deciding to invest in a stock. Many of the tools we used back then don't exist anymore, so I rebuilt the application to bring it back to life!

How it Works:

Visit this link, create an account and query any stock from the NASDAQ 100. Stocks that have been queried prior are cached in a database. New queries are done by first web scraping from Motley Fool or calling the Contextual News Search API, then running IBM Watson's NLP model on the text. Moola will display the stock chart, emotional score and sentiment score for each queried company. So what are you waiting for? It's Moola Time hehe!

Iris
Flask, Python, SQLite, Azure Cognitive Services

Objective:

Iris is an accessible web application for those who are visually impaired to "see" and understand surrounding content via crowd-sourced knowledge.

Motivation:

An ACM study in 2019 reported that the IBM Mobile Accessibility Checker found that 67% of apps from a random sample have accessibility issues. Accessibility issues are also not just limited to technology. Blog posts and other online accounts by visually impaired individuals express how every day experiences are challenges, as well, such as groceries, online shopping, banking, etc.

How it Works:

Our web application is a simple, accessible interface. Users take a picture of content they would like to know more about - e.g. a restaurant menu, a grocery price label, a metro sign - and upload it to our tool. We run the Azure Cognitive Services API to complete an initial classification of what the image portrays and send the image to our user-community for a more detailed review. These results are the reported back.

Snow Day Forecaster: A Machine-Learning-Based Snow Prediction System
Flask, Python, MongoDB, JQuery, Bootstrap

Objective:

The Snow Day Forecaster is web application that provides students living in my county a reliable guess on school closings based on the weather conditions.

Motivation:

The motivation behind this project was that my county's school board would often wait till 5 AM the next day before deeming whether schools should close or not, inconveniencing students, parents, and teachers alike.

How it Works:

This application was primarily built using Flask and MongoDB. It calls the Weather Underground and Twitter APIs for both historical weather forecasts and real-time weather data. Then, using scikit-learn, it inputs the weather conditions as a set of features to predict the chance of school-closings.

**This application boasts a 99% test accuracy.
